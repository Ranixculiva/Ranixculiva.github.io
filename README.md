# 餐廳菜單設計修改總結

## 概述
本次設計修改主要針對早午餐菜單進行重新規劃，提升客人體驗和視覺效果，並***提升營收***。

## 顯眼程度排序
按照重要性和視覺突出程度，各項目排序如下：

**套餐** > **蛋捲** > **炒蛋** > **奶油煎吐司** = **義大利麵** > **其他** >>>> **單點**

## 午餐菜單分區設計
因為單點利潤不高，我們希望盡量減少他在菜單上重要程度。所以我們決定拆掉單點區，並將各個分類重新分區，單點變成各區的加價購選項。
### A區：輕食區
- **主要內容**：經典漢堡、奶油煎吐司套餐、輕食主義套餐
- **加價購選項**：濃湯（價錢待定）、沙拉（45）、可樂餅(40)

### B區：重點區（名稱待定）
- **主要內容**：歐姆蛋捲拼盤、美式炒蛋小拼盤
- **加價購選項**：
  - 奶油餐包/香蒜麵包/薯餅/培根（統一價錢：30元）
  - 德腸（35元）
  - 牛肉排（55元）
- **設計要求**：要比其他區更顯眼

### C區：麵區
- **主要內容**：義大利麵系列
- **新增項目**：打拋肉醬麵系列
- **特色**：不需要提示加價購
- **設計要求**：可能需要新增一個板塊來呈現打拋系列 或是可能多一個空行 看怎麼呈現比較好

### D區：飲料區
- **新增項目**：牛乳（不計入套餐優惠，因為成本高）

#### 飲料分類：

1. **特級原片系列**（無糖）
   - 油切綠茶
   - 蜜香紅茶
   - 冷泡/熱沖

1. **英國早餐茶系列**（無糖）
   - 帝瑪Dilmah系列（澳洲熱門）
   - 紅茶、鮮奶茶
   - 僅限熱飲


1. **錫蘭紅茶系列**（半糖）
   - 紅茶、奶茶、鮮奶茶
   - 同茶基底
   - 可做冰溫熱 顯示方式可參考QBurger菜單

1. **風味飲品**
   - 豆漿、抹茶牛奶、可可牛奶
   - 冰溫熱

1. **現磨咖啡**
   - 美式、拿鐵咖啡、榛果拿鐵

**職人牛乳**（200ml）：單點不計入套餐 要再想一下如何呈現（主要是媽媽帶小朋友會點 老闆不希望他們帶seven的進來喝 垃圾還丟這 但基本上我們賣這利潤不高 所以不讓客人用套餐作折抵）

## 套餐區設計
- **視覺要求**：要有圖片（有可能我們要自己手繪或是找一些適合的圖），顯眼，主打 可參考Q burger 
- **重點**：省錢
- **備註**：要記得備注差額補差價

## 區分設計
### 早餐 vs 午餐
- **老闆想法**：參考能量小姐的押印設計
- **內容**：用英文寫 "Brunch"、"Breakfast"
- **可選元素**：加上時鐘顯示時間
- **目標**：或是有其他想法也行 反正讓客人容易區分
- **營業時間**：放在早午餐區塊

## 視覺設計要求
- **底色偏好**：老闆喜歡最後一張的綠色底色 希望早餐也使用相同的綠色底色
